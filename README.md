# 🎮 Повтори комбінацію

**Повтори комбінацію** — це класична гра на пам'ять, де ти повинен запам'ятати та повторити послідовність світлих кнопок з супроводжуючими звуками. Кожний раунд послідовність стає довшою — спроби впоратися зі все більшим викликом! 

## 🕹️ Як грати

1. **Натисни кнопку A**, щоб почати гру
2. **Спостерігай** за послідовністю — кнопки будуть світитися та звучати 👀
3. **Повтори комбінацію** в тому ж порядку, натискаючи кнопки
4. Якщо правильно — послідовність подовжується! ✅
5. Якщо помилився — гра закінчується ❌

## 🎯 Правила

- 📊 **Рахунок**: кількість правильно повторених послідовностей
- 🏆 **Найкращий результат**: найбільший рахунок за всі часи
- 🎨 **4 кнопки**: кожна має унікальний колір та звук
- ⏱️ **Без часових обмежень** — думай стільки, скільки потрібно!
